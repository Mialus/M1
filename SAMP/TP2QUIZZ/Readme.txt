Classe Main :

Cette classe est appelé au démarrage de l'application,
elle vous donne entre ajouter des questions et le quizz avec les questions actuelles.

Classe ajoutQuestion :

Cette classe permet d'ajouter et de supprimer une question :
	-pour ajouter, il suffit de taper la question, de choisir entre vrai ou faux, puis d'appuyer sur le bouton.
	-pour supprimer, il faut appuyer longtemps sur la question voulu.

Classe Quizz :

Cette classe gére le quizz.

Classe ReponseQizz :

Cette classe permet d'afficher la réponse à la question courante si l'utilisateur le veut.

Classe SQL :

Cette classe gére la base sql du quizz.
